The next generation DataHub. Turn Github into a DataHub. Share data + content in a useable form with team-mates or the world.

## Deploy

Check out the [datahub-next-deploy repo](https://github.com/datopian/datahub-next-deploy) for information about deploying this project.

## Dev

To setup a local dev environment:

1. Clone this repo
2. Install the dependencies with `npm i`
3. Run `npm run dev`
