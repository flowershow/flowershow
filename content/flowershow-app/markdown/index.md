# Markdown is Awesome

Markdown is the universal language of the web. You already use it — on GitHub, Reddit, Slack, Discord, ChatGPT, Notion, and more. It's the simplest way to write structured content that works everywhere.

## Why Markdown?

- **Simple**: Write naturally with a few symbols (`# heading`, `**bold**`, `[link](url)`). No complex editors, no proprietary formats.
- **Portable**: Markdown is plain text. It works on every platform, every device, every OS. Your content is never locked in.
- **Future-proof**: Plain text files will be readable in 50 years. Can you say that about your `.docx` files?
- **AI-friendly**: Every major AI tool — ChatGPT, Claude, Copilot — speaks Markdown natively. It's the lingua franca of the AI era.
- **Everywhere**: Google Docs now supports Markdown. Apple Notes added Markdown import/export. The world is converging on Markdown.

## What can you do with Markdown?

Build websites, blogs, documentation, knowledge bases, wikis, catalogs — all from plain text files you own. Combine Markdown with tools like Obsidian, GitHub, and [Flowershow](https://flowershow.app) and you have a powerful, free stack for publishing anything on the web.
