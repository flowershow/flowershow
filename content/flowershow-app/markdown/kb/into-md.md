## \# into.md

Convert any webpage into LLM-friendly markdown.

## \## Usage

Prefix the URL you want to convert with [into.md/](https://into.md/)

```
E.g.: into.md/
https://zod.dev/v4/changelog

         |                 |

      Add this          Your URL
```
