---
title: Getting Started — From the Terminal (CLI)
description: Publish any folder of markdown from the terminal. No repo required.
---

1. Install: download the `fl` binary from the [releases page](https://github.com/flowershow/flowershow/releases)
2. Log in: `fl login`
3. Publish: `fl ./my-folder`
4. Update later: `fl ./my-folder`

See [[reference/cli|CLI reference]] for all commands and options.
